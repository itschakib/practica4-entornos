Instalar Eclipse IDE

  1. Visita https://www.eclipse.org/downloads/
  2. Descarga “Eclipse IDE for Java Developers”.
  3. Instala con opciones por defecto.
  4. Abre Eclipse y configura tu espacio de trabajo.

Instalar Plugins

Para mejorar la experiencia, Eclipse permite añadir plugins. Ejemplo: EGit (para GitHub), Dark Theme, Lombok.
