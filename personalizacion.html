Personalización de Eclipse

Tema y Apariencia

Para cambiar el tema: Window → Preferences → General → Appearance → Theme.

Opciones de Menú Importantes

  • File: Crear, abrir o guardar proyectos.
  • Edit: Copiar, pegar, buscar.
  • Navigate: Moverse entre clases y métodos.
  • Run: Ejecutar programas Java.
  • Window: Ajustar la vista del entorno.

Depuración

Para depurar, establece breakpoints (clic en el margen izquierdo), luego ejecuta “Debug As → Java Application”.
